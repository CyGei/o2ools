# outbreaker2 toy dataset

The outbreaker2 result generated from the example in the [outbreaker2
vignette](https://www.repidemicsconsortium.org/outbreaker2/articles/introduction.html).
This dataset was produced by running `outbreaker()` on the
`fake_outbreak` data.

## Usage

``` r
out
```

## Format

An `outbreaker_chains` object.
