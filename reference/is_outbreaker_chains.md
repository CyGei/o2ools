# Check if an object is a data frame of class 'outbreaker_chains'

This function checks if the input is a data frame and of class
'outbreaker_chains'.

## Usage

``` r
is_outbreaker_chains(out)
```

## Arguments

- out:

  The object to be checked.

## Value

TRUE if the object is a data frame and of class 'outbreaker_chains',
otherwise stops with an error.
